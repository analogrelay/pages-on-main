Pages on Main

GitHub Pages only supports master and gh-pages right now. Until it supports other branches, if you want to change your default branch to something other than master you can use the workflow defined in this repository to mirror commits in your default branch to gh-pages.

Steps:

  1. Create a gh-pages branch if there isn't one already and push it to GitHub.
  2. Commit the .github/workflows/copy-pages.yml file from this repo to your `main` branch
